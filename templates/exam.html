Akses Kamera Dibutuhkan

Aplikasi ini memerlukan akses ke kamera laptop/PC Anda untuk pengawasan ujian terstandarisasi.

  • Klik ikon 🔒 **Gembok** di sebelah kiri URL pada address bar browser.
  • Cari pengaturan **Kamera** (Camera) dan ubah ke **Izinkan** (Allow).
  • Muat ulang halaman jika diminta, lalu klik tombol coba lagi di bawah.
  • Buka menu **Safari** -> **Settings for This Website...** di menu bar atas.
  • Cari baris **Camera** dan ubah pengaturannya menjadi **Allow**.
  • *Di iOS/iPadOS*: Buka Settings -> Safari -> Camera -> ubah menjadi **Allow**.
  • Klik ikon kamera/perizinan di sebelah kiri URL bar.
  • Klik tanda silang (x) di sebelah **Blocked Temporarily** untuk mereset izin.
  • Muat ulang halaman, lalu pilih **Allow** saat dialog perizinan muncul.
  • Masukkan alamat situs ini (misal: `http://192.168.100.49:5001`) ke setting insecure origins Chrome/Edge:
  • Buka tab baru ke `chrome://flags/#unsafely-treat-insecure-origin-as-secure`, masukkan domain di kolom input, ubah ke **Enabled**, dan klik **Relaunch**.
  • *Rekomendasi Utama*: Hubungi administrator atau jalankan server via HTTPS dengan parameter `--https` agar otomatis terenkripsi dan aman.

Setelah melakukan perbaikan di atas, klik tombol di bawah untuk mengaktifkan kembali kamera.
